Instructions

This app is for downloading data from the database.

Data is downloaded in the form of a .csv file.

As there are no guaranties that strings in the database does not contain commas, some editors may have a problem reading in the file properly. Excel usually do a good job of reading in the data correctly.

 

There are four versions of download.

 

  1. Download all data.

Downloads everything there is in the database

 

  1. Download by DOI number

An easy way to get all data from one paper, or for checking to see if your paper is in the database

 

  1. Download selected data

Provides the same filtering options as in the interactive graphics but download all categories for the selected cells and not only the ones used in the interactive graphics.

 

  1. Download by list of database ID

Askes for a .txt-file with comma separated numbers corresponding to database ID numbers and download data for those cells.
