Instructions

The purpose of this plot is to compare the how the outdoor stability has evolved over time, but there are plenty of options to compare different parameters against each other.

 

Redirection to original sources

By clicking on a datapoint in the figure, you are directly redirected to the original paper. This is based on link generated from the DOI number and works most of the time.

 

Using the figures

The figures generated with this tool can be downloaded and you are free to use them any way you want, but please, cite the source.

 

Categorical plots

The default behaviour when changing what is compared on the x-axis is to filter out the five categories with most data. Otherwise, you may end up with a plot of a few thousand categories, which both is hard to interpret and time consuming to generate. After the initial canvas is drawn, the categories can be selected freely.

Values of -1 in the data table

The framework used for generating the figures does not handle nan values very well. All nan values have therefore been converted to -1 before plotting. This is only the case for data found in the data table and not for downloaded data.

 

Concerning behaviour of sliders

When both endpoints of a range slider are at the end points, it is not used while filtering the data, i.e., data beyond the slider limits are included.

 

Concerning PCE values

For the PCE values in the figure. If a stabilised value is available, that one is used. If not, the value from the reversed voltage scan is used (i.e. Voc to 0) if available. If not, values from the forward scan are used.

 

Concerning light intensity

In the datasets, there are cells that have been measured at non-standard condition e.g. with indoor lightning or under monochromatic lightning. Those cells could easily get efficiencies higher than what they would get under standard condition. To exclude those cells, the standard procedure is to filter out any cell where the reported light intensity is not between 90 and 110 mW/cm^2. That can easily be changed by using the light intensity slider.

 

Concerning options in menus

The options in the multiselect menus are generated automatically to include every option recorded in the database. That means that there could be more than one formatting variation for the same thing if someone has missed to follow all formatting instructions when uploading the data to the database.

 

Replotting of figures

Operations under the headline “Figure properties” redraws the figure which could be a slow operation when a large number of points must be generated.

Buttons with Boolean filters and multiselect menus may require additional data to be read from the database but does not require the figure to be redrawn which usually is a faster operation.

 

Downloading data

When downloading data from the data table, a csv file is generated with all data in the database for the selected cells. If a large number of cells is chosen, this could be a slow process. Excel usually does a good job reading in the generated CSV file with care taken to corner cases like data fields containing commas or other special characters.
